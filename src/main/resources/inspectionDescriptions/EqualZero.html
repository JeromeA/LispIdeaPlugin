Reports the use of "=" to compare a value to zero.

This inspection reports the use of the equality operator to compare a value to zero. This is inefficient and can be confusing. It's better to use the function ZEROP.

Examples:

  // Inefficient:
  (when (= myvalue 0)
        (println "myvalue is zero"))

  // Better:
  (when (zerop myvalue)
        (println "myvalue is zero"))
